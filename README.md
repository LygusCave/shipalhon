# SHIPALHON 
* Shipalhon (白파란红) is a CJK to russian translitiration systems translator
* It's created for translators, to hlep them, if they're forget/don't know specific system.
* Right now, there is only CN to Palladius' system translator, created with using a ['pinyin-pro'](https://github.com/zh-lx/pinyin-pro). And JP to Polivanov's system translator, created via ['kuroshiro'](https://github.com/hexenq/kuroshiro) and ['kuroshiro-analyzer-kuromoji'](https://github.com/hexenq/kuroshiro-analyzer-kuromoji). 
* In the futere, I'm planning to add and Kontsevich's system for KR.
