# SHIPALHON 
* Shipalhon (白파란红) is a CJK to russian translitiration systems translator
* It's created for translators, to hlep them, if they're forget/don't know specific system.
* Right now, there is only CN to Palladius' system translator, created with using a '@pinyin-pro' and '@pinyin-pro/data'(CompleteDict)
* In the futere, I'm planning to add the Polivanov's system for JP and Kontsevich's system for KR.
